Homework 6

This is my implementation of the Game of Life broken down into 4 layers of development. I was able to create a partially functional game using object oriented javascript programming and canvas graphics. A lot of time was spent developing the object oriented structure of the program so that the cells on the grid would be represented as members of a class with specific functions. One of the hardest ideas to implement was checking all neighbors without index errors or double-checking cells. In its current state there are still some logical bugs with the rules of the game. A feature that I am proud of is the dynamically resizeable canvas which has width and height sliders that can be set freely. I also implemented "play" and "stop" functionality and some extra buttons to randomize cells or invert their status. The canvas can also be cleared, setting all cells to dead. This assignment involved a culmination of JavaScript skills working together with HTML and CSS elements. I got in depth with Java classes and functions to complete this assignment.
